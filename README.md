## Installation

OpenAgentic is compatible with all major Node.js package managers:

**npm:**
```bash
npm install openagentic
```

**pnpm:**
```bash
pnpm add openagentic
```

**yarn:**
```bash
yarn add openagentic
```

**bun:**
```bash
bun add openagentic
```
